## ğŸ¯ MISSION
You are building a **modern web application** for an existing job scraper system. This web app will:
1. Display all scraped jobs in a beautiful, filterable interface
2. Allow the user to mark jobs as Applied/Interested/Not Interested
3. Show a dashboard with statistics and insights
4. Provide settings to adjust scoring weights and preferences
5. Sync new jobs from GitHub Actions artifacts
6. Run locally first (with optional Docker setup)

**CRITICAL**: You are in **AGENT MODE**. Build the complete web app, test it, refine it, and ensure it works end-to-end. Make reasonable decisions and document them.

---

## ğŸ“‹ CONTEXT: Existing System

The user already has a **working job scraper** that:
- Runs daily via GitHub Actions
- Scrapes 3 APIs: Bundesagentur, Arbeitnow, RemoteOK
- Stores results in SQLite (`jobs.db`)
- Sends Telegram notifications
- Uploads `jobs.db` as GitHub artifact

**Your job**: Build a web interface to VIEW, FILTER, and MANAGE these jobs.

## ğŸ¨ UI/UX DESIGN SPECIFICATIONS

### Design System: Minimalist & Clean

**Color Palette:**
```css
/* Primary */
--background: #ffffff
--foreground: #0a0a0a

/* Scores */
--score-excellent: #10b981  /* Green: 9-10 */
--score-good: #3b82f6       /* Blue: 7-8.9 */
--score-okay: #f59e0b       /* Amber: 5-6.9 */
--score-low: #9ca3af        /* Gray: <5 */

/* Status */
--status-applied: #8b5cf6    /* Purple */
--status-interested: #06b6d4 /* Cyan */
--status-not-interested: #ef4444 /* Red */

/* UI Elements */
--border: #e5e7eb
--muted: #f3f4f6
--accent: #3b82f6
```

**Typography:**
- Font: Inter (clean, modern, readable)
- Headers: Font weight 600-700
- Body: Font weight 400
- Code/Numbers: Font weight 500

**Spacing:**
- Generous whitespace (padding: 24px-32px for containers)
- Consistent 8px grid system

**Components:**
- Rounded corners: 8px (cards), 6px (buttons), 4px (inputs)
- Subtle shadows: No heavy drop-shadows, use border + light shadow
- Hover states: Slight background change + border color change
- Transitions: 150ms ease-in-out

**Icons:**
- Use Lucide React (minimalist icon set)
- Size: 20px (default), 16px (small), 24px (large)

```python
from sqlalchemy import create_engine

import os

DATABASE_URL = os.getenv("DATABASE_URL", "sqlite:///./jobs.db")

engine = create_engine(
    DATABASE_URL,
    connect_args={"check_same_thread": False}  # SQLite specific
)

SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)
Base = declarative_base()
```

```python
# app/models/job.py
from pydantic import BaseModel
from typing import Optional, List
from datetime import datetime
from enum import Enum

class UserStatus(str, Enum):
    NOT_APPLIED = "not_applied"
    APPLIED = "applied"
    INTERESTED = "interested"
    NOT_INTERESTED = "not_interested"
    ARCHIVED = "archived"

class JobBase(BaseModel):
    title: str
    company: str
    location: Optional[str]
    url: str
    platform: str
    description: Optional[str]
    salary_min: Optional[int]
    salary_max: Optional[int]
    salary_currency: Optional[str]
    posted_date: Optional[str]
    money_score: float
    passion_score: float
    location_score: float
    total_score: float

class Job(JobBase):
    id: int
    job_hash: str
    scraped_at: datetime
    user_status: UserStatus = UserStatus.NOT_APPLIED
    user_notes: Optional[str]
    user_updated_at: Optional[datetime]
    
    class Config:
        from_attributes = True

class JobUpdate(BaseModel):
    user_status: Optional[UserStatus]
    user_notes: Optional[str]

class JobQueryParams(BaseModel):
    page: int = 1
    limit: int = 20
    sort: str = "total_score"
    order: str = "desc"
    minScore: Optional[float] = None
    maxScore: Optional[float] = None
    locations: Optional[List[str]] = None
    companies: Optional[List[str]] = None
    platforms: Optional[List[str]] = None
    status: Optional[UserStatus] = None
    dateFrom: Optional[str] = None
    dateTo: Optional[str] = None
    search: Optional[str] = None
```

**Jobs Endpoint:**
```python
# app/routes/jobs.py
from fastapi import APIRouter, Depends, Query
from sqlalchemy.orm import Session
from typing import List, Optional
from app.database.connection import get_db
from app.models.job import Job, JobQueryParams, JobUpdate
from app.services.job_service import JobService

router = APIRouter(prefix="/api/jobs", tags=["jobs"])

@router.get("/", response_model=dict)
async def get_jobs(
    page: int = Query(1, ge=1),
    limit: int = Query(20, ge=1, le=100),
    sort: str = Query("total_score"),
    order: str = Query("desc"),
    minScore: Optional[float] = Query(None, ge=0, le=10),
    maxScore: Optional[float] = Query(None, ge=0, le=10),
    locations: Optional[str] = Query(None),  # Comma-separated
    companies: Optional[str] = Query(None),
    platforms: Optional[str] = Query(None),
    status: Optional[str] = Query(None),
    dateFrom: Optional[str] = Query(None),
    dateTo: Optional[str] = Query(None),
    search: Optional[str] = Query(None),
    db: Session = Depends(get_db)
):
    """Get paginated, filtered jobs"""
    params = JobQueryParams(
        page=page,
        limit=limit,
        sort=sort,
        order=order,
        minScore=minScore,
        maxScore=maxScore,
        locations=locations.split(",") if locations else None,
        companies=companies.split(",") if companies else None,
        platforms=platforms.split(",") if platforms else None,
        status=status,
        dateFrom=dateFrom,
        dateTo=dateTo,
        search=search
    )
    
    service = JobService(db)
    result = service.get_jobs(params)
    
    return {
        "jobs": result["jobs"],
        "total": result["total"],
        "page": page,
        "limit": limit,
        "totalPages": (result["total"] + limit - 1) // limit
    }
```

**GitHub Sync Service:**
```python
# app/services/github_sync.py
import httpx
import os
from datetime import datetime
from app.database.connection import SessionLocal

class GitHubSyncService:
    def __init__(self):
        self.token = os.getenv("GITHUB_TOKEN")
        self.repo = os.getenv("GITHUB_REPO")  # e.g., "username/job-scraper"
    
    async def download_latest_artifact(self):
        """Download latest jobs.db from GitHub Actions artifacts"""
        if not self.token or not self.repo:
            raise ValueError("GitHub token or repo not configured")
        
        headers = {
            "Authorization": f"Bearer {self.token}",
            "Accept": "application/vnd.github+json"
        }
        
        async with httpx.AsyncClient() as client:
            # 1. Get latest workflow run
            url = f"https://api.github.com/repos/{self.repo}/actions/runs"
            response = await client.get(url, headers=headers)
            runs = response.json()["workflow_runs"]
            
            if not runs:
                raise ValueError("No workflow runs found")
            
            latest_run = runs[0]
            
            # 2. Get artifacts from this run
            artifacts_url = latest_run["artifacts_url"]
            response = await client.get(artifacts_url, headers=headers)
            artifacts = response.json()["artifacts"]
            
            # 3. Find jobs-database artifact
            db_artifact = next(
                (a for a in artifacts if a["name"] == "jobs-database"),
                None
            )
            
            if not db_artifact:
                raise ValueError("jobs-database artifact not found")
            
            # 4. Download artifact
            download_url = db_artifact["archive_download_url"]
            response = await client.get(download_url, headers=headers)
            
            # 5. Extract and save (it's a zip file)
            import zipfile
            import io
            
            with zipfile.ZipFile(io.BytesIO(response.content)) as z:
                z.extract("jobs.db", path="./temp/")
            
            return "./temp/jobs.db"
    
    async def import_jobs(self, db_path: str):
        """Import jobs from downloaded DB into main DB"""
        import sqlite3
        
        # Connect to downloaded DB
        temp_conn = sqlite3.connect(db_path)
        temp_cursor = temp_conn.cursor()
        
        # Get all jobs
        temp_cursor.execute("SELECT * FROM jobs WHERE sent_to_user = 1")
        jobs = temp_cursor.fetchall()
        
        # Import into main DB
        db = SessionLocal()
        imported = 0
        
        for job_data in jobs:
            # Check if job already exists (by job_hash)
            existing = db.execute(
                "SELECT id FROM jobs WHERE job_hash = ?",
                (job_data[1],)  # job_hash is column 1
            ).fetchone()
            
            if not existing:
                # Insert new job
                db.execute(
                    """INSERT INTO jobs (...) VALUES (...)""",
                    job_data
                )
                imported += 1
        
        db.commit()
        temp_conn.close()
        
        # Clean up temp file
        os.remove(db_path)
        
        return imported
```

## ğŸš¨ COMMON PITFALLS TO AVOID

1. **CORS Issues**
   - Add CORS middleware to FastAPI
   - Allow localhost:3000 origin during development

2. **Database Schema Mismatch**
   - Ensure backend schema matches scraper's schema
   - Add new columns (user_status, etc.) with migrations
   - Don't break existing scraper functionality

3. **Large Payloads**
   - Don't send full descriptions in job list
   - Use pagination (20 per page)
   - Load full details only when needed

4. **Filter Performance**
   - Add database indexes on commonly filtered columns
   - Do filtering in SQL, not in Python/JS

5. **Date Formatting**
   - Use ISO 8601 format for API (YYYY-MM-DD)
   - Use date-fns for display formatting
   - Handle timezones correctly

6. **GitHub API Rate Limits**
   - Cache artifact downloads
   - Don't sync too frequently
   - Handle 403 errors gracefully

7. **State Management**
   - Don't duplicate state between components
   - Use React Query for server state
   - Invalidate queries after mutations
